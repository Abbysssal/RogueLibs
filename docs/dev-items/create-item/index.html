Creating a Custom Item

CustomItem class#

To make a custom item, you need to create a class deriving from CustomItem:

MyCustomItem.cs
public class MyCustomItem : CustomItem{}

There's only one method that you need to implement - SetupDetails:

MyCustomItem.cs
public class MyCustomItem : CustomItem{    public override void SetupDetails()    {        Item.itemType = ItemTypes.Tool;        Item.itemValue = 200;        Item.initCount = 1;        Item.rewardCount = 1;        Item.stackable = true;        Item.hasCharges = true;    }}

This method is called only once, when the item is created or spawned.
See more info later on this page.

By default, your custom item will have the same name as the class (Type.Name).
You can change that behavior with an ItemName attribute:

MyCustomItem.cs
[ItemName("My_Custom_Item")]public class MyCustomItem : CustomItem{    ...}

Initialization#

Just call the following method in your plugin's Awake():

MyCoolPlugin.cs
public void Awake(){    RogueLibs.CreateCustomItem<MyCustomItem>();}

You can set your item's name and description using WithName and WithDescription methods:

MyCoolPlugin.cs
public void Awake(){    RogueLibs.CreateCustomItem<MyCustomItem>()        .WithName(new CustomNameInfo("My Custom Item"))        .WithDescription(new CustomNameInfo("My Custom Item is very cool and does a lot of great stuff"));}

You can do the same with sprites and unlocks:

MyCoolPlugin.cs
public void Awake(){    RogueLibs.CreateCustomItem<MyCustomItem>()        .WithName(new CustomNameInfo("My Custom Item"))        .WithDescription(new CustomNameInfo("My Custom Item is very cool and does a lot of great stuff"))        .WithSprite(Properties.Resources.MyCustomItem)        .WithUnlock(new ItemUnlock { UnlockCost = 10, CharacterCreationCost = 5 });}
