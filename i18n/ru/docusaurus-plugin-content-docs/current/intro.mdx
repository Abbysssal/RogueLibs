# Введение

RogueLibs - библиотека для моддинга Streets of Rogue.

Код игры довольно плохой и если вы попытаетесь написать мод для неё самостоятельно, у вас выйдет це-е-елая куча патчей! Дайте RogueLibs сделать все нужные патчи, а **вы просто сконцентрируйтесь на контенте!**

Зацените [гайд по установке](user/installation)!

Тут также есть [гайд для разработчиков](setup)!
