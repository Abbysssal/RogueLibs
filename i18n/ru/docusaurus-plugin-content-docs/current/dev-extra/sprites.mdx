## Добавляем спрайты как ресурсы {#sprites-as-resources}

Для начала, конвертируйте спрайты в PNG или JPEG, если они не PNG или JPEG.

Потом, измените расширение файлов на `.bin`. Таким образом Visual Studio определит файлы как двоичные данные вместо растрового изображения.

:::note
Нужен скриншот
:::

Кликните ПКМ по проекту и нажмите **Свойства** в самом низу:

<img src={useBaseUrl('/img/dev-extra/sprites/project-properties.png')} width='300'/>

Перейдите во вкладку **Ресурсы** и нажмите на *вон ту* кнопку чтобы создать файл ресурсов:

<img src={useBaseUrl('/img/dev-extra/sprites/create-resources.png')} width='800'/>

Выберите **Добавить ресурс** на верху меню и нажмите **Добавить существующий файл...**:

<img src={useBaseUrl('/img/dev-extra/sprites/add-resource.png')} width='600'/>

Теперь найдите свои изображения (с расширением `.bin`) и добавьте их как ресурсы:

:::note
Нужен скриншот
:::

Теперь у вас должно получиться использовать данные ваших спрайтов в коде вот так: `Properties.Resources.<НазваниеРесурса>`.

:::note
Если высветилась ошибка, убедитесь, что пространство имён по умолчанию, указанное в свойствах проекта во вкладке **Приложение**, такое же, как и то, которое вы используете.
:::

## Инициализация {#initialization}

Обычно, новые `RogueSprite` создаются так:

```csharp
RogueLibs.CreateCustomItem<MyCustomItem>()
    ...
    // highlight-next-line
    .WithSprite(Properties.Resources.MyCustomItem);
```

Вы можете создавать их напрямуют тоже, но вам придётся указать их название и область применения. Область применения спрайта определяет область игры, в которой спрайт будет доступен. `SpriteScope.Items` работает только на предметах, `SpriteScope.Objects` - только на объектах, и т.д. Вы можете указать несколько областей объединив их оператором `|`.

```csharp
RogueLibs.CreateCustomSprite("name", SpriteScope.Items, Properties.Resources.ResourceName);
```

По умолчанию, RogueLibs использует весь файл для текстуры. Вы можете указать область текстуры, которую вы хотите использовать, параметром типа `Rect`:

```csharp
RogueLibs.CreateCustom("name", SpriteScope.Items, Properties.Resources.ResourceName,
                                // highlight-next-line
                                new Rect(32f, 32f, 64f, 64f));
```

Если вы будете использовать текстуры размера не 64x64, то вам также надо будет указать параметр PPU (pixels-per-unit):

```csharp
RogueLibs.CreateCustom("name", SpriteScope.Items, Properties.Resources.ResourceName,
                                // highlight-next-line
                                new Rect(64f, 64f, 128f, 128f), 128f);
```
